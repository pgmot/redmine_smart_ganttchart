redmine_smart_ganttchart
===

## Overview
Improve Redmine gantt chart.

## Features
- Display date on the top of the table
- Edit start date and due date with drag and drop

## Installation
### 1. Clone
```sh
$ cd (redmine root)/plugins
$ git clone https://github.com/mizoguche/redmine_smart_ganttchart
```

### 2. Restart Redmine
Restart your redmine
